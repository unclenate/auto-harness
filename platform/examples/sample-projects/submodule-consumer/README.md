# submodule-consumer — sample project

This is a **demonstration** of what a consumer repository looks like after it
integrates auto-harness via the *git submodule* topology. It is a static
artifact — nothing here is intended to be cloned and run as-is.

## What this sample shows

In the submodule topology, auto-harness lives inside the consumer repo at a
fixed mount point:

```text
.harness/                              # git submodule → auto-harness
  platform/
    core/kernel/base/
    agents/base/
    agents/claude-code/
    skills/harness-governance/
    templates/
    validators/
```

The consumer repo owns everything *outside* `.harness/`: its own source code,
its `harness.manifest.yaml`, its governance artifacts under `docs/**`, and
thin bridge files (`CLAUDE.md`, `AGENTS.md`, `HARNESS.md`) that tell AI
assistants where to look.

## Why `.harness/` is missing here

This sample does **not** ship a real git submodule — it is an educational
snapshot of the *shape* of a consumer repo. The `.harness/` directory does
not exist on disk, which means:

- `.claude/skills/harness-governance` is a **dangling-by-design** relative
  symlink pointing at `../../.harness/platform/skills/harness-governance`.
- `.agents/skills/harness-governance` is the same, for non-Claude agents.

In a real consumer these resolve correctly once `git submodule update --init`
populates `.harness/`. Keeping the symlinks dangling here preserves the exact
on-disk layout readers will create in their own repos.

## Topology-agnostic manifests (ADR-0003)

The `harness.manifest.yaml` in this sample is **byte-identical** to the one in
`../node-web-saas-postgres/harness.manifest.yaml`. The manifest declares
*which modules* are active; it never declares *where the harness is mounted*.
You can verify:

```bash
diff ../node-web-saas-postgres/harness.manifest.yaml ./harness.manifest.yaml
# (no output — files are identical)
```

This is the central claim of ADR-0003: integration topology (subtree vs.
submodule vs. vendored copy) is decoupled from the governance manifest.
