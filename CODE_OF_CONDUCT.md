# Code of Conduct

The auto-harness project has adopted the **Contributor Covenant Code of Conduct, version 2.1**, as our community standard. The full canonical text is published and maintained by the Contributor Covenant project at:

> [https://www.contributor-covenant.org/version/2/1/code_of_conduct/](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)

We adopt that text by reference. Adoption by reference is explicitly endorsed by the Contributor Covenant project and is used by many open-source projects to keep their local Code of Conduct in step with upstream updates.

---

## Scope

This Code of Conduct applies in all project spaces — including issues, pull requests, discussions, code reviews, and any communication channel operated by the project. It also applies when an individual is officially representing the auto-harness project in public spaces.

---

## Reporting

If you experience or witness conduct that does not align with the standards described in the Contributor Covenant v2.1, please report it privately:

**Contact:** Nate DiNiro — nate@bdits.io

When sending a report, please include:

- A description of what happened
- Where it happened (issue / PR / discussion link)
- When it happened (approximate date is fine)
- Any other context that would help understand the situation

All reports will be reviewed and handled confidentially. The maintainer commits to acknowledging receipt within three business days.

---

## Enforcement

Enforcement follows the guidelines published in the Contributor Covenant v2.1, available at the URL above. Outcomes may include private feedback, a temporary cooling-off period from project spaces, or — in serious or repeated cases — permanent removal from project spaces.
